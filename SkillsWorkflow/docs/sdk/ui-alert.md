# Alert

A sub namespace of UI, exclusive for alert operations

```javascript
//accessing to ui.alert methods
SW.UI.Alert.{methodName}
```

---

## clear

#### Description

This method can be used to clear a toastr that is being displayed.

#### Method(s)

```js {3}
1 function clear(): void
```

#### Basic Usage

```javascript
>    SW.UI.Alert.clear(),
```
