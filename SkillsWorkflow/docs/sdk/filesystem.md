# FileSystem

The namespace FileSystem provides to developers a method to easily upload files to a folder.

```javascript
//accessing to FileSystem methods
SW.FileSystem.{methodName}
```

---

## uploadFilesToFolder

#### Description

This method can be used to upload files to a folder.
